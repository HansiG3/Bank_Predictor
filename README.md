# Customer Churn Prediction System

A machine learningâ€“based web application that predicts whether a bank customer is likely to churn (leave the bank) based on demographic and financial features.

---

## Features
- Predicts customer churn using a trained neural network
- Uses real-world bank customer dataset
- Encodes categorical features using saved encoders
- Flask-based backend for easy deployment
- Model and encoders loaded dynamically for inference

---

## Tech Stack
- **Programming Language:** Python
- **Machine Learning:** TensorFlow / Keras
- **Data Processing:** Pandas, NumPy
- **Model Serialization:** Pickle (`.pkl`)
- **Web Framework:** Flask
- **Dataset:** Bank Customer Churn Dataset
